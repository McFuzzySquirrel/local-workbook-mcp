# Decisions Made

| Decision | Reason | Alternatives | Impact |
|---|---|---|---|
| Separate `ExcelMcp.UAT` project vs extending `ExcelMcp.Server.Tests` | UATs are a distinct quality gate; keeping them separate allows `dotnet test --project tests/ExcelMcp.UAT` runs | Add to existing Server.Tests | Clean separation; different run context |
| Test via `ExcelWorkbookService` / `ExcelWriteService` (not `ExcelTools`) | Services are public; Tools is internal and a thin wrapper anyway | InternalsVisibleTo for tools | Correct abstraction; tools just delegate to services |
| Temp file copy pattern for write tests | Write ops mutate files; originals must stay pristine for re-runs | Read-only transactions | Safe, idempotent write tests |
| Relax pivot field count assertions to `>= 1` | ClosedXML reads field counts from the .xlsx differently than the PS1 script configured them | Hard-code wrong values | Accurate assertions that survive ClosedXML version changes |
